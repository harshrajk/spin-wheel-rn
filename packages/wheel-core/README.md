# @acme/wheel-core

Pure TypeScript utilities for spin wheel validation, winner resolution, and rotation planning.

## Install

```bash
npm install @acme/wheel-core
```

## Exports

- `validateSegments`
- `resolveWinnerIndex`
- `buildSectorGeometry`
- `planRotation`
- `createSeededRng`
- all primary types (`WheelSegment`, `SpinRequest`, `SpinResult`)
