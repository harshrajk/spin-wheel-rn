# Demo Expo (Expo go testing)

This app folder is for testing the spin wheel component on iOS and Android.

## Setup

```bash
npm install
```

## Run

From the root directory:
```bash
npm start -w examples/expo-demo
```

Then:
- Press `i` on macOS to open iOS simulator
- Press `a` to open Android emulator
- Scan QR code to open on physical device
